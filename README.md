# Smart Manufacturing CAPEX Dashboard

This repository presents a Smart Manufacturing CAPEX Dashboard developed in Power BI to centralize capital expenditure tracking, improve financial visibility, and enable data-driven decision-making across projects.

**Note:** All screenshots are blurred to maintain confidentiality.

---

## 1. Application Overview

The Smart Manufacturing CAPEX Dashboard enables stakeholders to monitor planned vs actual capital expenditures, track project performance, and support data-driven financial decision-making across manufacturing initiatives.

### Key Enhancements

1. **Centralized Financial View:**
  Consolidates CAPEX data from multiple sources into a single Power BI dashboard, eliminating fragmented reporting.

2. **Plan vs Actual Tracking:**
  Provides clear visibility into planned vs actual CAPEX, enabling better budget control and variance analysis.

3. **Data Integration:**
  Integrated multi-source financial datasets significantly improve data consistency and reliability.

4. **Efficiency Improvement:**
  Reduced reconciliation effort by **75%** by automating data consolidation and reporting workflows.

5. **Interactive Filtering:**
  Dynamic filters (Year, Quarter, Workstream, Project) allow users to drill down into specific segments for deeper analysis.

6. **Real-Time Insights:**
  Enables near real-time tracking of financial performance, supporting faster and more informed decision-making.

7. **Project-Level Visibility:**
  Displays detailed project-level metrics including P&L impact, headcount planning, and inventory-related investments.
